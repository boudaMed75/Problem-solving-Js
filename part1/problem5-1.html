. Créer un script qui demande à l’utilisateur de saisir trois variables numériques via une boite prompt. L’utilisateur est averti s’il ne saisit pas un nombre, et le script lui demande de saisir une valeur numérique. Une fois que l’utilisateur saisit trois valeurs numériques, le script teste les trois variables :

  • Si elles sont égaux, écrire : "Les trois variables sont identiques."
  • Si seulement deux sont identiques, écrire :« Deux des variables sont de valeurs égales."
  • Sinon, écrire :"Les trois variables sont différentes
